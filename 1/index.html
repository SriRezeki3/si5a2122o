    1.Pada pertemuan pertama Rekayasa web dijelaskan bagaimana cara mendapatkan CMS wordpress.
    2.Dijelaskan juga bagaimana cara menginstal wordpress.
    - pertama jika belum mendownload wordpress, buka di google id.wordpress.org.
    - Kedua, jika sudah muncul scroll ke bawah sampai muncul tulisan install wordpress.
    - Jika sudah didownload buka ekstrak file wordpress tersebut.
    - Jika sudah di extract, ubah nama wordpress menjadi si5a2122o.
    - selanjutnya, file yang sudah di ubah menjadi si5a2122o di copy ke htdocs.
    3.Di pertemuan ini diajarkan juga bagaimana cara membackup file si5a2122o.
    - Folder bikin pertama SI5A - 1.
    - selanjutnya copy file si5a2122o yang ada di htdocs ke folder SI5A - 1.
    - Selanjutnya ekspor file si5a2122o di database. Jika sudah letakan file di folder SI5A - 1. Jadi di dalam folder SI5A - 1 ada si5a2122o dan ql.
    4. Jika ingin merestore, buka impor di phpadmin lalu pilih file ql tadi, setelah itu tunggu sampe selesai.
